СОЗДАНИЕ ШАБЛОНА


1. Создай папку с названием шаблона в Template/
2. Создай файл body.x (Это файл куда ты будешь подключать дальнейшие части шаблона) и head.x (там все заголовки и подключение стилей производится штатными способами)
3. Подключать файлы нужно в body.x вот так : <!--load:{filename}.x;-->
4. В файл index.php нужно лезть только чтобы поменять заголовок (Скоро допилю многостраничность чтобы изменять заголовки для каждой страницы)
5. Чтобы изменить шаблон, нужн зайти в админ панель. Логин admin и все. и там выбирай.
